# UnityBowlingProjectExecutableOnly

## Download
To download the project, click the green code button on the homepage of this repo.\
Then select download zip. Once the zip is downloaded, go to its location in file explorer.\
Right click on the folder, and select extract. Specify where you would like the folder to be\
extracted (desktop is a good spot, or a games folder). To play the game, open the extracted\
folder and double click on the executable, titled Bowling Mini Project. Happy gaming!

Currently includes a working bowling game with high score system.
